# 🌤️ Clima Agora - Dashboard Meteorológico

Aplicação que consome dados de uma API de meteorologia para fornecer informações precisas sobre as condições climáticas de qualquer cidade em tempo real.

## ✨ Funcionalidades
- **Busca por Cidade:** Digite o nome da cidade para obter os dados atuais.
- **Dados Detalhados:** Exibição de Temperatura, Umidade, Pressão Atmosférica e Vento.
- **Interface Dinâmica:** O fundo do site ou ícones mudam conforme o estado do tempo (Sol, Chuva, Nuvens).

## 🛠️ Tecnologias
- **HTML5 / CSS3** (Layout responsivo)
- **JavaScript ES6+** (Consumo de API com Fetch)
- **OpenWeatherMap API** (Fonte de dados)

## 📡 Conceitos Aplicados
Este projeto foi fundamental para praticar:
- Requisições assíncronas (`Promises` e `Async/Await`).
- Tratamento de erros (ex: quando a cidade não é encontrada).
- Manipulação de objetos JSON complexos.
